Request / Filter Data Flow
01
FilterBar.vue

→
02
useFilters()
Singleton composable updates shared reactive refs
→
03
View Component
Watches filter changes, calls api.js method
→
04
api.js
Builds query params, fires axios GET request
→
05
FastAPI Route
apply_filters() + filter_by_month() on in-memory data
→
06
Pydantic
Validates & serialises response model
→
07
Vue Reactivity
Computed properties update, DOM re-renders
Global Filter Dimensions
FILTER Time Period — month (Jan–Dec) or quarter (Q1–Q4), maps to ?month=
FILTER Warehouse — San Francisco / London / Tokyo, maps to ?warehouse=
FILTER Category — Circuit Boards / Sensors / Actuators / Controllers / Power Supplies
FILTER Order Status — Delivered / Shipped / Processing / Backordered
